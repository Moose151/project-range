Add Range Dashboard to your desktop

Give users one-click access to this dashboard from their desktop. The download creates a launcher named Range Dashboard that opens this site at its current address. For the SEW Range logo to appear as the app icon, use Install app below.

Windows

Downloads Range Dashboard.url. Save it to your Desktop and double-click to open the dashboard.

Download Windows shortcut
Ubuntu / Linux

Downloads Range Dashboard.desktop. Move it to your Desktop, then right-click → Allow Launching (or chmod +x) to enable double-click.

Download Linux launcher
Install app (recommended — shows the logo icon)

In Chrome or Edge, open the browser menu and choose Install Range Dashboard (or the install icon in the address bar). This adds a real app window to the desktop / Start menu / dock with the SEW Range logo as its icon.

Note: the Install app option only appears when the site is served over HTTPS or opened on the host machine itself (localhost). On a plain-HTTP LAN address, use the downloadable shortcuts above.
